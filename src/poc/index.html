This website is currently under construction. The aim of the website is to create an interactive tutorial for the well known Chrome developer tools where user will guided to use the tool step by step.
Differently from other tutorials, we are trying to force the user to interact with the F12 dev tool to progress into the course.
We are still gathering ideas, so if you have an idea or would like to cooperate, please feel free to send me an email here or contribute to the gitHub project at this link
